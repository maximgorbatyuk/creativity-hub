Projects Dashboard

Manage all your creative projects in one place. Filter by status, track progress, and switch between active, completed, and archived projects at a glance.

Ideas Capture

Save ideas with automatic source detection for Instagram, TikTok, Pinterest, YouTube, and websites. Never lose inspiration again.

Notes

Write and organize notes for each project. Keep context, references, and thoughts close to the work they belong to.

Checklists

Break projects into actionable steps with checklists. Track progress with completion indicators and link items to work logs.

Expenses & Budget

Track project expenses by category. Monitor budgets with filtering, sorting, and multi-currency support to keep spending under control.

Documents

Store project files including PDFs, images, and documents. Preview files in-app and keep everything organized per project.

Reminders

Set reminders for deadlines, follow-ups, and important dates. View upcoming reminders on the Today dashboard so nothing slips through.

Work Logs

Log time spent on project tasks. Link work entries to checklist items and see duration summaries to understand where your time goes.

Global Search

Search across all projects, ideas, notes, checklists, and expenses. Find anything instantly with full-text search and quick navigation.
