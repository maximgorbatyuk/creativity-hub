# App Store Releases

Use this file for App Store Connect "What's New" text.

## 2026.1.1

### English

```
We are launching CreativityHub — a single place to manage creative projects with ideas, notes, checklists, expenses, documents, reminders, and work logs. We also added global search, sharing from other apps, and optional iCloud backups, so your workflow stays organized and your data stays easy to restore.
```

### Russian

```
Мы запускаем CreativityHub — единое пространство для творческих проектов с идеями, заметками, чеклистами, расходами, документами, напоминаниями и логами работы. Также добавили глобальный поиск, отправку данных из других приложений и резервные копии в iCloud, чтобы вам было проще держать все в порядке и безопасно восстанавливать данные.
```
